# Laravel AI Cognee

Cognee integration for Laravel AI: RAG, conversation store, and content ingestion.

## Features

- Conversation store that pipes transcripts to Cognee for semantic search
- RAG context provider (fluxui-chat) for document search
- Memory context provider (fluxui-chat) for cross-conversation recall
- SearchDocumentsTool and SearchConversationMemoryTool for Laravel AI agents
- Configurable dataset resolver for multi-tenant RAG
